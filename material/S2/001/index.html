    Clase 1: Clase de Repaso de Python¶

    Strings¶

    Los strings (cadenas de texto) son secuencias de caracteres inmutables. Pueden incluir letras, símbolos, números y espacios. Se declaran con comillas simples 'texto', dobles "texto" o triples '''texto''' para varias líneas.

    Se pueden manipular con operaciones como:

    • Concatenación: unir dos strings. Ejemplo: "Hola" + " Mundo"
    • Repetición: repetir un string varias veces. Ejemplo: "a" * 3 → "aaa"
    • Acceso a caracteres: con índices. Ejemplo: "Hola"[1] # 'o'
    • Slicing: subcadenas. Ejemplo: "Hola"[1:3] # "ol"
    s = "Hola Mundo"
print(s.upper())   # 'HOLA MUNDO'
print(s.lower())   # 'hola mundo'

    s = "  texto con espacios  "
print(s.strip())  # 'texto con espacios'
print(s.lstrip()) # elimina izquierda
print(s.rstrip()) # elimina derecha

    s = "banana"
print(s.replace("a", "o"))   # 'bonono'
print("ana" in s)            # True
print(s.count("a"))          # 3

    s = "uno,dos,tres"
partes = s.split(",")       # ['uno', 'dos', 'tres']
print("-".join(partes))     # 'uno-dos-tres'

    Listas¶

    Las listas son estructuras de datos ordenadas, mutables y heterogéneas, es decir, pueden almacenar elementos de distintos tipos (enteros, strings, booleanos, otras listas, entre otros.). Se definen usando corchetes [] o el constructor list() y los elementos están separados por comas.

    numeros = [1, 2, 3]
numeros.append(4)      # [1, 2, 3, 4]
numeros.insert(1, 99)  # [1, 99, 2, 3, 4]

    lista = [1, 2, 3, 2]
lista.remove(2)    # elimina el primer 2 → [1, 3, 2]
lista.pop()        # elimina el último → [1, 3]

    valores = [4, 1, 3, 2]
valores.sort()             # [1, 2, 3, 4]
valores.reverse()          # [4, 3, 2, 1]
print(valores.count(3))    # 1

    lista = [10, 20, 30]
suma = sum(lista)       # 60
maximo = max(lista)     # 30
minimo = min(lista)     # 10

    Estructura de control if¶

    La instrucción if permite ejecutar código sólo si se cumple una condición.
    Puede complementarse con elif (condición alternativa) y else (caso por defecto).

    Las condiciones se evalúan como valores booleanos (True o False).

    Ejemplo de sintaxis:

    x = 15
if x < 10:
    print("Menor que 10")
elif x == 10:
    print("Es 10")
else:
    print("Mayor que 10")

    ==   # Igualdad
!=   # Diferente
>    # Mayor que
<    # Menor que
>=   # Mayor o igual
<=   # Menor o igual

    and  # Ambas condiciones deben ser verdaderas
or   # Al menos una condición debe ser verdadera
not  # Niega una condición

    edad = 20
nota = 85

    if edad >= 18 and nota >= 70:
        print("Aprobado como adulto")
    else:
        print("No cumple los requisitos")

    for loop¶

    El bucle for permite recorrer secuencias como listas, cadenas o rangos, y ejecutar un bloque de código para cada elemento.
    Es una herramienta fundamental para procesar datos en colecciones.

    Ejemplo básico
    for letra in "Hola":
    print(letra)

    Ejemplos comunes¶

    Iterar sobre una lista
    frutas = ["manzana", "pera", "uva"]
for fruta in frutas:
    print(fruta)
    Usar range()
    for i in range(3, 8):
    print(i)  # Imprime del 3 al 7
    Acceder a índices con enumerate()
    colores = ["rojo", "verde", "azul"]
for indice, color in enumerate(colores):
    print(f"{indice}: {color}")
    Iterar sobre un string
    nombre = "python"
for caracter in nombre:
    print(caracter.upper())

    while loop¶

    El bucle while permite repetir instrucciones mientras se cumpla una condición lógica. Se utiliza cuando no se conoce de antemano cuántas repeticiones se deben realizar.

    Bucle descendente
    contador = 3
while contador > 0:
    print(contador)
    contador -= 1

    Consideraciones importantes¶

    • Si la condición nunca deja de cumplirse, el bucle será infinito.
    • Se puede usar break para salir del bucle en un momento determinado.
    • También se puede usar continue para saltar a la siguiente iteración sin ejecutar el resto del bloque.
    while True:
    entrada = input("Escriba 'fin' para salir: ")
    if entrada == "fin":
        break

    numero = 0
while numero < 5:
    numero += 1
    if numero == 3:
        continue  # salta el 3
    print(numero)

    Tuplas¶

    Una tupla es una estructura de datos similar a una lista, pero inmutable. Sus elementos no pueden ser modificados una vez creada. Se define con paréntesis () o con el constructor tuple().

    Las tuplas son útiles para representar datos agrupados que no deben cambiar, como coordenadas, fechas o pares clave-valor temporales.

    Ejemplo básico
    punto = (3, 4) # Inmutable
x, y = punto
print(x)  # 3
print(y)  # 4

    Propiedades importantes¶

    tupla = ("lunes", "martes", "miércoles")
print(tupla[0])  # lunes

    dias = ("lunes", "martes")
# Error: las tuplas no se pueden modificar
dias[0] = "domingo"

    uno = (5,)  # la coma es necesaria
print(type(uno))  # <class 'tuple'>

    Diccionarios¶

    Un diccionario es una colección de pares clave-valor. Permite almacenar y acceder a valores usando claves personalizadas, como strings o números.

    Se define con llaves {}, y cada par clave-valor se separa por :.

    Ejemplo básico
    persona = {
    "nombre": "Ana",
    "edad": 30
}

print(persona["edad"])  # 30

    Operaciones comunes¶

    persona["edad"] = 31           # Modifica edad
persona["correo"] = "a@b.com"  # Agrega nueva clave

    for clave, valor in persona.items():
    print(clave, "->", valor)

    if "nombre" in persona:
    print("Nombre registrado")

    del persona["edad"]

    Funciones¶

    Una función es un bloque de código reutilizable que realiza una tarea específica. Permite organizar el código, evitar repeticiones y facilitar la lectura.

    Se define con la palabra clave def, se le da un nombre, se indican los parámetros entre paréntesis y se coloca el bloque indentado.

    Definición básica
    def saludar(nombre):
    return f"Hola, {nombre}"

print(saludar("Luis"))  # 'Hola, Luis'

    Otras posibilidades¶

    Función sin parámetros
    def mostrar_menu():
    print("1. Iniciar")
    print("2. Salir")
    Función con varios parámetros
    def sumar(a, b):
    return a + b
    Valores por defecto
    def saludar(nombre="invitado"):
    return f"Hola, {nombre}"
    Funciones que no retornan (procedimientos)
    def imprimir_saludo(nombre):
    print(f"Hola, {nombre}")

    Recursividad¶

    La recursividad es una técnica en la que una función se llama a sí misma para resolver un problema dividiéndolo en subproblemas más pequeños. Cada llamada reduce el problema hasta llegar a un caso base, que detiene la recursión.

    Ejemplo clásico
    def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

    Consideraciones importantes¶

    # Toda función recursiva debe tener una condición de salida
# que evite llamadas infinitas

def cuenta_regresiva(n):
    if n == 0:
        return
    print(n)
    cuenta_regresiva(n - 1)

    # Olvidar el caso base puede generar un error por límite de recursión

def infinito():
    return infinito()  # Error: RecursionError
