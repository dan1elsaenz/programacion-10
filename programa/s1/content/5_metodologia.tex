\section{Metodología}

El curso se desarrollará combinando clases magistrales con espacios de práctica supervisada.
Se hará uso extensivo del lenguaje de programación Python mediante un entorno de desarrollo integrado, con énfasis en la edición, ejecución, prueba y depuración de programas.

A lo largo del semestre, se trabajará con ejercicios guiados y tareas orientadas a la resolución sistemática de problemas, para reforzar el diseño de algoritmos, el uso correcto de estructuras de control y el modelado con colecciones de datos.
Además, se incorporará práctica dirigida para la preparación de la Olimpiada Costarricense de Informática (OCI), mediante la solución y discusión de problemas de dificultad progresiva.
