\section{Objetivos}

\subsection*{Objetivo general}

Proveer formación básica en programación, construcción de algoritmos y de programas, para la resolución de problemas, por medio del lenguaje de programación Python.

\subsection*{Objetivos específicos}

Al finalizar este curso, la persona estudiante será capaz de:

\begin{enumerate}
  \item Diseñar, organizar e implementar algoritmos para resolver problemas específicos, por medio del aprendizaje de una lógica de programación adecuada.
  \item Usar un entorno de programación integrado para la edición, prueba y depuración de programas.
  \item Reutilizar componentes de código para la creación de software escalable y eficiente.
  \item Aplicar buenas prácticas de construcción de software.
  \item Aprender sobre los algoritmos y tipos de datos que se evalúan en la Olimpiada Costarricense de Informática (OCI).
\end{enumerate}
