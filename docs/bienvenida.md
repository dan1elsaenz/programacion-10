# :material-home-outline: Bienvenida

Este sitio web contiene el material del curso de programación del **Colegio Científico de Puriscal para 10° año**.
Aquí se encuentran los contenidos teóricos, enunciados de los laboratorios que se desarrollan en clase, las tareas y una sección especial para quienes deseen profundizar en la Olimpiada Costarricense de Informática (OCI).


> Se recomienda a las personas estudiantes utilizar este sitio como **referencia** durante el curso.
